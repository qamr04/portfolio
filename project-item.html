AWS Honeypot Project

Terug naar projects

  • Over dit project

    Dit project omvat het opzetten van een complete honeypot infrastructuur in Amazon Web Services (AWS). Het doel was om cyberaanvallen te detecteren, analyseren en te leren van de tactieken die aanvallers gebruiken.

    Een honeypot is een beveiligingsmechanisme dat opzettelijk kwetsbaar wordt gemaakt om aanvallers aan te trekken en hun activiteiten te monitoren. Door dit systeem in de cloud te implementeren, kon ik realistische aanvalsscenario's observeren en documenteren.

    Technologieën

    • AWS EC2

    • CloudWatch

    • Python

    • Linux

    Implementatie

    De honeypot werd geïmplementeerd met de volgende componenten:

    • EC2 Instance: Ubuntu server met opzettelijk kwetsbare diensten
    • Security Groups: Specifieke firewall regels om traffic te controleren
    • CloudWatch Logging: Realtime monitoring van alle activiteiten
    • Python Scripts: Automatische analyse van aanvalspatronen

    Resultaten

    Het systeem heeft succesvol verschillende aanvalspatronen gedetecteerd en gedocumenteerd. Door de data-analyse heb ik inzicht gekregen in veelvoorkomende aanvalsvectoren en beveiligingslekken.

    Belangrijkste bevindingen:

    • Gemiddeld 50+ aanvalspogingen per dag
    • Meest voorkomende aanvallen: SSH brute-force attacks
    • Geografische spreiding van aanvallers wereldwijd
    • Identificatie van automatische bot-netwerken
