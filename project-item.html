AWS Honeypot Project

Over dit project

Dit project omvat het opzetten van een complete honeypot infrastructuur in Amazon Web Services (AWS). Het doel was om cyberaanvallen te detecteren, analyseren en te leren van de tactieken die aanvallers gebruiken.

Technologieën

  • AWS EC2
  • CloudWatch
  • Python
  • Linux

Resultaten

Het systeem heeft succesvol verschillende aanvalspatronen gedetecteerd en gedocumenteerd. Door de data-analyse heb ik inzicht gekregen in veelvoorkomende aanvalsvectoren en beveiligingslekken.
